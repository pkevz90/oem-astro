Instructions
  • Building a plan
    • Importing satellites with STK report
      • STK Arts Report should be within scenario
      • Within STK, run report on desired origin satellite, then select desired deputy satellite
      • Satellite names for both the chief and deputy are imported, if not origin satellite exists, one will be created with correct name.
      • Select entire line at report timestep and copy line
      • Paste into either the satellite menu if no satellite exists, or update state context menu
      • System will check if names on satellite match the name from the report and origin sat. If does not match, user given chance to override
      • Satellite will be updated with new state, burns in future will be recalculated to account for new state
      • Other satellites in scenario will be propagated to the time frame specified by the import
      • Sim time and Sun vector are also imported from STK
      • Importing group of satellites with J2000 states
      • Format of Epoch Date X Y Z Vx Vy Vz in J2000
      • Line above should start with properties, then name, shape, color, accelartion, etc.
      Importing as one group of satellites with J2000 states
      • STK J2000 Position Velocity Report
      • Choose all satellites you want in scenario on left side
    • Add Satellite by right clicking into satellite menu, then entering either with RMOE or cartesian values
    • By default, all burns are computed with finite accelerations, goto for impulsive burns in 1000 mm/s2
    • To insert a burn, click and hold current satellite position, use slider to change shown time
    • Burn planning options
      • Click and drag to desired waypoint, changing time of flight with mouse wheel
      • Switch to manual burn direction on right-click menu to click and drag burn direction
      • Open burns panel by right-clicking satellite itself and manually insert RIC coordinate and TOF
      • More burn options available within right-click menu
      • Particle Swarm Optimization utilized to find best position to gain sun, based on delta-V
    • Burn Right Click
      • Right clicking on a burn opens up context menu with options
      • Right clicking also copies to clipboard text string for ingestion into STK
      • By default right-clicking shows you burn magnitudes in RIC and allows alteration of burn direction
      • Shift right-clicking brings up burn waypoint information and allows alteration of waypoint
    • Burns can be deleted by ctrl-clicking on the burn point
    • Limitations
      • Burns durations are limited to 6 hours for mathematical stability
      • Burns are limited to once every 30 minutes
      • Two-body dynamics are utilized, therefore an error of ~100m per hour can be expected compared to full relative dynamics
  • Spacebar changes current view
  • GIF Generation
    • Select options panel
    • Set start time and end time (in minutes from start of scenario)
    • Select time step per frame (assuming 30 FPS, so if time step is 1 minute, each hour will be 2 seconds long in video)
    • Press record
